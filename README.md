NextStep
NextStep is an interactive Android application designed to help teenagers explore their interests, personality traits, and career awareness through engaging mini-games, quizzes, and multiplayer challenges. The platform transforms self-discovery into a fun and social experience using timed activities, personality assessments, and friendly competition.

Overview
NextStep combines personality-based quizzes, skill-oriented mini-games, and multiplayer interactions to create an engaging environment where users can learn more about themselves while having fun. The application focuses on encouraging curiosity, self-reflection, and social learning without forcing predefined career choices.
